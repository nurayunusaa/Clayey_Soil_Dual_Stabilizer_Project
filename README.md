# Clayey Soil Stabilization Project

This repository contains data, scripts, supplementary materials, and manuscript files for the study:

**Title:** Multiscale Analysis of Clayey Soil Behavior Treated with Dual Stabilizers: Experimental and Predictive Modeling Approaches (Wood Ash and Sodium Chloride)

## Structure

- `main_paper/`: Manuscript, figures, and tables for the main paper.
- `supplementary/`: Supplementary tables and figures.
- `data/`: Raw and processed datasets.
- `scripts/`: Python scripts for preprocessing, ML modeling, and figure generation.
- `notebooks/`: Jupyter notebooks for exploration, modeling, and figure generation.
